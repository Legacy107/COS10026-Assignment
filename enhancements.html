Enhancements

Responsive Design

Various techniques including the use of media queries, flexbox and grid are utilized to make the page responsive to different screen layouts e.g. desktop, tablet and mobile layout. By doing that, the website can be accessed on a variety of devices which enhances user's experience.

Advanced CSS properties and selectors

Throught out all pages, animation and transition properties are applied on different element in order to increase the interactivity as well as maintain a smooth and aesthetically pleasing experience.

Parallax effect

At the top of the topic page, the parallax effect is applied on the image element so that it is scrolled at a different pace compared to other elements. This is achieved using 3D transformation. The parallax effect help creating a nicer scrolling experience which enhances the overall user's experience of the topic page.
